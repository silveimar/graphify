| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-33-18 | Tampering | `to_obsidian()` community label injection | mitigate | Use validated `ConceptName.title` and existing template sink sanitizers; explicit caller `community_labels` remains a controlled override. |
| T-33-19 | Tampering | dry-run sidecar behavior | mitigate | Pass `dry_run` to `resolve_concept_names()` and assert `concept-names.json` is absent after dry-run. |
| T-33-20 | Tampering | template sink rendering | mitigate | Keep sink-specific sanitization in `safe_filename`, `safe_tag`, `_sanitize_wikilink_alias`, and frontmatter dumping; add combined generated-title regression coverage. |
| T-33-21 | Elevation of Privilege | CLI parser positional confusion | mitigate | Remove parsed repo identity flags from positional args before target selection; missing values exit 2. |
