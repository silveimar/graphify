    2. `list_snapshots(root: Path = Path(".")) -> list[Path]` — returns `sorted(snapshots_dir(root).glob("*.json"), key=lambda p: p.stat().st_mtime)`. Returns empty list if no snapshots.

    3. `save_snapshot(G: nx.Graph, communities: dict[int, list[str]], root: Path = Path("."), name: str | None = None, cap: int = 10) -> Path`:
       - Generate timestamp: `datetime.now(timezone.utc).strftime("%Y-%m-%dT%H-%M-%S")`

       - Write to `.tmp` then `os.replace(tmp, target)` (per cache.py::save_cached pattern)
       - FIFO prune: `snaps = sorted(snapshots_dir(root).glob("*.json"), key=lambda p: p.stat().st_mtime); for p in snaps[:-cap]: p.unlink()`

       - Read and parse JSON
       - Deserialize graph: try `json_graph.node_link_graph(payload["graph"], edges="links")` with TypeError fallback
       - Communities: `{int(k): v for k, v in payload["communities"].items()}`
       - Return `(G, communities, payload.get("metadata", {}))`
       - Raise `ValueError` on missing keys or corrupt JSON

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| user input → snapshot filename | --name flag value from CLI becomes part of filesystem path |
| disk → snapshot load | JSON files in snapshots/ could be hand-edited or corrupted |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-01 | Tampering | snapshot.py save_snapshot | mitigate | Atomic write via os.replace(tmp, target) prevents partial writes on crash |
| T-06-02 | Tampering | snapshot.py load_snapshot | mitigate | Validate required keys ("graph", "communities", "metadata") exist; raise ValueError on missing |
| T-06-03 | Information Disclosure | snapshot.py --name flag | mitigate | Sanitize name with re.sub(r'[^\w-]', '_', name)[:64] — no path traversal possible |
| T-06-04 | Denial of Service | snapshot.py disk growth | mitigate | FIFO prune on every save_snapshot call; cap defaults to 10 |
| T-06-05 | Elevation of Privilege | snapshot.py path confinement | accept | Snapshots write only to graphify-out/snapshots/ which is already in .gitignore; no user-controlled base path |
</threat_model>
