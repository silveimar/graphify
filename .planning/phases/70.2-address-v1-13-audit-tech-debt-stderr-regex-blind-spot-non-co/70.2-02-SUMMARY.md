# Phase 70.2 Plan 02: SCHEMA_VERSION single source of truth — Summary

One-liner: Moved `schema_version` literal `"1.13"` into a single `SCHEMA_VERSION` constant in `graphify/build.py`, stamped on every graph at construction time so MCP/query/downstream callers see it without going through export.

## What Changed

- **graphify/build.py**: Added `SCHEMA_VERSION = "1.13"` near top of module. Stamped `G.graph["schema_version"] = SCHEMA_VERSION` at end of both `build_from_json()` and `build()`.
- **graphify/export.py**: Added `from .build import SCHEMA_VERSION`. Replaced literal `"1.13"` fallback in the schema_version write with the imported constant.
- **tests/test_build.py**: Added 4 tests covering the constant value lock, both build paths' stamping, and export.py's reuse of the constant.
