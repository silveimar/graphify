# Phase 08.1 Plan 02: Approve & Pipeline Wiring — Snapshot Auto-Wire Summary

**One-liner:** Wired `auto_snapshot_and_delta` into both full-pipeline and cluster-only skill.md paths so every `/graphify` run automatically saves a snapshot and generates GRAPH_DELTA.md.

## What Was Done

Closed INT-02: snapshots and deltas previously required a separate `graphify snapshot` command. After this plan, every `/graphify` invocation calls `auto_snapshot_and_delta` immediately after `to_json`, in both the full pipeline path and the cluster-only path.

### Task 1: Insert auto_snapshot_and_delta calls into skill.md

Added two identical blocks to `graphify/skill.md`:

**Full pipeline path** (after `to_json` at ~line 425):
```python
from graphify.snapshot import auto_snapshot_and_delta
_pre_snap_count = len(list(Path('graphify-out/snapshots').glob('*.json'))) if Path('graphify-out/snapshots').exists() else 0
_snap_path, _delta_path = auto_snapshot_and_delta(G, communities, root=Path('.'))
if _pre_snap_count > 0:
    print(f'Snapshot saved ({G.number_of_nodes()} nodes). Delta: see GRAPH_DELTA.md')
else:
    print(f'Snapshot saved ({G.number_of_nodes()} nodes). No prior snapshot for delta comparison.')
```

**Cluster-only path** (after `to_json` at ~line 962): identical block.
