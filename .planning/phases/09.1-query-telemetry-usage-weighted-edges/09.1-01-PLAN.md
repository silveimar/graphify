    ```python
    def _check_derived_edges(
        G: nx.Graph,
        telemetry: dict,
        out_dir: Path,
        agent_edges: list[dict],
    ) -> None:
        """Propose 2-hop shortcut edges when traversal threshold met. Per D-06/D-07/D-08."""
        threshold = telemetry.get("threshold", 5)
        counters = telemetry.get("counters", {})
        existing = {(e["source"], e["target"]) for e in agent_edges}
        existing |= {(e["target"], e["source"]) for e in agent_edges}
        added = False
        for key_ab, count_ab in list(counters.items()):
            if count_ab < threshold:
                continue
            a, b = key_ab.split(":", 1)
            if b not in G:
                continue
            for neighbor in G.neighbors(b):
                key_bc = f"{min(b, neighbor)}:{max(b, neighbor)}"
                if counters.get(key_bc, 0) < threshold:
                    continue
                pair = (a, neighbor)
                rpair = (neighbor, a)
                if pair in existing or rpair in existing:
                    continue
                if G.has_edge(a, neighbor):
                    continue
                record = {
                    "source": a,
                    "target": neighbor,
                    "relation": "derived_shortcut",
                    "confidence": "INFERRED",
                    "confidence_score": 0.7,
                    "source_file": "",
                    "via": b,
                    "traversal_count": min(count_ab, counters[key_bc]),
                }
                agent_edges.append(record)
                existing.add(pair)
                added = True
        if added:
            _save_agent_edges(out_dir, agent_edges)
    ```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| MCP query -> telemetry recording | Edge keys derived from graph node IDs (already sanitized at extraction time) |
| telemetry.json -> derived edges | Derived edges constrained to nodes already in G.neighbors() |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-09.1-01 | Tampering | telemetry.json | accept | File contains only node ID pairs (already sanitized by security.py at extraction) and integer counters. No user-controlled input reaches edge keys. |
| T-09.1-02 | DoS | telemetry.json size | accept | Counter dict bounded by graph edge count (finite). No per-query logging per D-01. |
| T-09.1-03 | Tampering | derived edges via crafted queries | mitigate | _check_derived_edges uses G.neighbors() to constrain targets to valid graph nodes only; cannot inject arbitrary edges |
| T-09.1-04 | Information Disclosure | telemetry.json exposes query patterns | accept | File is local to graphify-out/, same access level as graph.json itself |
