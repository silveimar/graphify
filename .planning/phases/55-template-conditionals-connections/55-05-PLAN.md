    **Required structure (each `##` heading + one annotated fence per D-55.11):**

    ```markdown
    # Graphify Template Reference

    Authoritative reference for the block-template surface in `string.Template`-based rendering. The engine lives in `graphify/templates.py` (Phase 31, extended in Phase 55). Block syntax expands BEFORE `${}` substitution; outputs route through label/HTML sanitization.

    ## Conditional blocks

    Wrap content in `{{#predicate}}...{{/if}}`. The opening tag chooses one of:
    - A built-in predicate (see [Predicate catalog](#predicate-catalog))
    - `{{#if_attr_<name>}}` — escape hatch reading `node[<name>]` directly
    - `{{#if_note_type_<X>}}` — true iff render_note received `note_type=X` (Phase 55)
    - `{{#if_flag_<name>}}` — profile-declared flag (see [Authoring predicate_flags:](#authoring-predicate_flags))

    Pitfall: blocks are NOT nestable. `validate_template` rejects nested `{{#...}}{{#...}}` at preflight.

    ```template <!-- test:conditional-basic -->
    Title
    {{#if_note_type_thing}}This is a thing note.{{/if}}
    ```

    ## Connection loops

    Iterate outbound/inbound edges with `{{#connections}}…{{/connections}}`. Inside the loop, `${conn.<field>}` and `${conn_<field>}` are bound to per-edge fields. Order is deterministic: `(relation, label)`.

    ```template <!-- test:connections-loop -->
    Connections:
    {{#connections}}- ${conn.relation} → ${conn.label}
    {{/connections}}
    ```

    Available fields: `relation`, `label`, `target_id`, `confidence`. Missing fields render as empty strings (sanitized).

    Pitfall: empty connection lists render NOTHING (the loop body is dropped). Phase 31 test `test_connections_empty_loop_renders_nothing` enforces.

    ## Ordering invariant

    **Block expansion runs BEFORE `${}` substitution (D-16).** This means a `${var}` reference inside a `{{#if_*}}` body is substituted only if the block evaluates true; conversely, you cannot use `${...}` to construct a block opener at runtime — that text would expand AFTER blocks are processed.

    ```template <!-- test:ordering-invariant -->
    {{#if_note_type_thing}}NAME: ${name}{{/if}}
    ```

    ## Sanitization

    Every value reaching the rendered text passes through `_sanitize_wikilink_alias` or equivalent. New Phase 55 predicates emit booleans only — no string output, no new sanitization sink.

    ```template <!-- test:sanitization -->
    {{#connections}}- ${conn.label}
    {{/connections}}
    ```

    Even when an edge target's label contains `<script>...</script>`, the rendered output is HTML-escaped. T-31-01 contract.

    ## Predicate catalog

    | Predicate | Semantics |
    |-----------|-----------|
    | `if_god_node` | True iff the node is a top-K god node |
    | `if_isolated` | True iff the node has zero edges |
    | `if_dataview_nonempty` | True iff the Dataview block is non-empty |
    | `if_attr_<name>` | True iff `node[<name>]` is truthy (escape hatch) |
    | `if_note_type_<X>` | True iff `note_type==X`; X ∈ {thing, statement, person, source, code, moc} |
    | `if_flag_<name>` | True iff `<name>` is declared in `predicate_flags:` and evaluates True |

    ```template <!-- test:predicate-catalog -->
    {{#if_note_type_thing}}thing-only content{{/if}}
    ```

    ## Authoring `predicate_flags:`

    Top-level profile.yaml key. Maps a flag name to a node-attribute rule. Two rule shapes:

    ```yaml
    predicate_flags:
      is_published: { attr: is_published }              # truthy
      is_reviewed:  { attr: status, equals: done }      # equality
    ```

    Rules: name must match `/^[a-z][a-z0-9_]*/`, must not collide with `_PREDICATE_CATALOG`, must not start with `attr_`, max 64 entries (T-55-FLAG-VAL DoS cap).

    ```template <!-- test:predicate-flags -->
    {{#if_flag_is_published}}PUBLISHED{{/if}}
    ```

    Reference flags via `{{#if_flag_<name>}}`. Distinct from `{{#if_attr_<name>}}` — the latter reads any attribute without profile declaration.

    ## Validation behavior

    `validate_profile_preflight` (Layer 2) calls `validate_template` for every template. Phase 55 also threads `known_flag_predicates=frozenset(profile['predicate_flags'])` so unknown `if_flag_*` references are rejected at preflight, not at render time.

    `graphify doctor` and `--validate-profile` surface the resulting errors. **Fallback contract (D-55.14):** when a template fails validation, the engine falls back to the builtin template and warns to stderr — no abort, no non-zero exit.

    ```template <!-- test:validation -->
    {{#if_note_type_thing}}ok{{/if}}
    ```

    ## Backward compatibility

    Block-free templates render byte-identically to v1.6 output. ROADMAP Phase 31 criterion 4 is enforced by `tests/test_templates.py::test_block_free_template_renders_byte_identical`. Adding new fields to `BlockContext` (Phase 55: `note_type`, `flag_predicates`) is non-breaking because both fields default to safe values.

    ```template <!-- test:backward-compat -->
    Static title with no blocks
    ```

    The above example contains no `{{#...}}` markers; the rendered output equals the template text verbatim (modulo `${...}` substitution).
