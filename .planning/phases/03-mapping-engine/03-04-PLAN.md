functions and assert they produce valid output without raising. Register `classify`,
`MappingResult`, and `validate_rules` in `graphify/__init__.py`'s lazy import map so

they catch any key-name mismatch between what classify() emits and what render_note()
consumes, BEFORE Phase 5's integration work starts.

tests in `tests/test_mapping.py`.
