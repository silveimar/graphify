## Tech Debt Summary

**3 cross-phase warnings:**

1. **`build.py:251` stderr non-compliance (AUDIT-02)** — emits `[graphify] Extraction warning (...)` which violates the D-04 three-prefix whitelist (`error:`, `info:`, `hint:`). Invisible to `test_stderr_contract.py` because the grep regex `print\([^)]*file=sys\.stderr` stops at the first `)` inside the f-string's `({len(real_errors)} issues)`. Two fixes recommended: (a) rename the prefix to `info:`/`error:`, and (b) tighten the invariant test regex to handle nested parens (e.g. require balanced `(...)` or use Python AST instead of grep).

2. **In-memory `schema_version` gap (CCONF-05 / CFED-03)** — `export.py::to_json` stamps `schema_version=1.13`, but `build.py::build_from_json` and `build.py::build` never set `G.graph['schema_version']`. Any future code reading the in-memory graph's version gets `None`. One-line fix in `build_from_json` and `build` resolves it.

3. **VFIX-02 doc precedence drift** — runtime chain in `__main__.py` is `--output > profile > option-b > --obsidian-dir > legacy default`, but README, `--help`, and 7 skill files omit the option-b step. Functional behavior is correct; docs are imprecise.
