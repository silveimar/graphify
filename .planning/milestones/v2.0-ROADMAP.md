1. **Temporal edge validity** — `valid_from` / `valid_until` / `decay_weight` on every edge; INFERRED-only supersession stamping with global tuple match and history retention; 4-site `analyze.py` filter excludes superseded edges from god-node and surprising-connection scoring; Temporal Health subsection in `GRAPH_REPORT.md`; Historical relations second-pass in wiki articles.
2. **Reasoning-relation edge types** — Five new typed relations on document/concept nodes (`supports`, `contradicts`, `supersedes`, `evolved_into`, `depends_on`) with code-node rejection; ADR-aware extraction prompts replicated across 10 platform skill files (PROMPT_VERSION 1.14.0); `contradictions_and_chains` analysis; typed wikilinks in Obsidian frontmatter distinguishable from structural relations.
3. **Content-fingerprint DEDUP spike** — Measured 18.78% aggregate near-duplicate concept-node rate across a code repo + doc-heavy corpus (PDF corpus unmeasured with documented rationale); resolves `Q-2026-05-07-01`; ship recommendation issued — actual implementation deferred to a future milestone per the measurement-only gate.
4. **Vault-CWD argparse fix** — `update-vault` and `vault-promote` exit-2 defect resolved (`--vault required=False` + tightened post-parse guard); 14-branch parametrized regression suite in `tests/test_vault_cwd_gate.py`; debug session `vault-cwd-gate-argparse-required.md` flipped to `status: resolved, resolved_in: phase-74-vbug`.
5. **graphifyy 2.0.0 PyPI release stamp** — Coordinated bump dance across `pyproject.toml`, `server.json` (manifest hash `1bc87657…`), and 12 platform `.graphify_version` stamps; D6 two-atomic-commit topology (`chore(75):` then `docs(75):`); PKG-02 pytest-matrix gate satisfied via D3 CI-as-contract (3.10 + 3.12).
