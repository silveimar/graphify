## New Finding: WIRING-01 (Low Severity)

**Location:** `graphify/__main__.py` line 843
**Affected:** TRIP-02 (degraded only when `--out-dir` is non-default)

`_approve_and_write_proposal` hardcodes manifest path to `Path("graphify-out") / "vault-manifest.json"`. When `--out-dir` is overridden to a non-default value, the manifest is loaded from the wrong path. `_load_manifest` degrades gracefully to `{}` (empty manifest = all notes treated as unmodified), so the default case works correctly but non-default `--out-dir` silently skips user-modified detection.

**Impact:** Functional for all default usage. Non-default `--out-dir` via `graphify approve` only degrades silently (no error, no data loss).
