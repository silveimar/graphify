- Iterate `proposals_dir.glob("*.json")`, parse each, collect into list
- Skip files that fail JSON parsing
- Return list sorted by timestamp ascending

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07-07 | Tampering | suggested_folder path traversal | mitigate | `sanitize_label()` applied to suggested_folder at proposal time; `validate_vault_path()` applied at approval time (Plan 03). Double defense. |
| T-07-08 | Tampering | proposal filename injection | mitigate | Filename is always `{uuid4}.json` — server-generated, never from agent input. Title stored only inside JSON payload. |
| T-07-09 | Spoofing | agent writes vault note without approval | mitigate | Tool writes ONLY to `graphify-out/proposals/`. No vault path accepted as tool argument. Vault write requires explicit `graphify approve --vault`. |
| T-07-10 | Injection | control characters in body_markdown | mitigate | `sanitize_label()` strips control characters from all string fields before persistence |
</threat_model>
