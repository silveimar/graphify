## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| MCP client → `call_tool` | Untrusted tool name/args; handlers already enforce sanitization |
| Workspace `graphify-out/*` → manifest | User-controlled sidecar files; manifest describes surface, not secrets |

## STRIDE

| Threat ID | Category | Component | Disposition | Mitigation |
|-----------|----------|-----------|-------------|------------|
| T-13-01 | Information disclosure | `capability_describe` live block | mitigate | Emit only aggregate stats / mtimes / non-secret ids — no annotation text, no peer_id (MANIFEST-03 scope) |
| T-13-02 | Tampering | `graphify-out/manifest.json` vs committed `server.json` | mitigate | `--validate` compares hashes; MCP server computes hash from **in-process** registry state, not trusting manifest.json alone for tool behavior |
| T-13-03 | Denial of service | Hash recompute every RPC | mitigate | Mtime-gated cache per D-04; no full-disk read per call |
</threat_model>
