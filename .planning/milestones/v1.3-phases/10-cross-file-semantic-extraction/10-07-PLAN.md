<objective>
Wire dedup provenance (D-15) through to the Obsidian vault adapter so that existing user wikilinks pointing at pre-merge node IDs continue to resolve after dedup. When `--obsidian-dedup` is enabled, every canonical node emits its eliminated IDs as Obsidian `aliases:` frontmatter entries; wikilinks like `[[auth]]` remain valid even after `auth` merged into `authentication_service`.

Purpose: Prevent the Phase 8 Obsidian adapter from silently breaking user vaults after dedup renames node IDs. Obsidian's native `aliases:` frontmatter field is the correct mechanism (it's how Obsidian itself forward-maps renamed notes), so this integration uses the existing Obsidian UX without inventing new concepts.

Output: render_note emits `aliases:` frontmatter when `merged_from` is set; to_obsidian hydrates `merged_from` from dedup_report.json when `obsidian_dedup=True`; `--obsidian-dedup` CLI flag propagates the option.

Scheduled in Wave 3 (after 10-04) because both this plan and 10-04 modify `graphify/__main__.py`; sequential ownership prevents merge conflicts.
</objective>

    ```python
    def _hydrate_merged_from(G: "nx.Graph", output_dir: Path) -> None:
        """D-15: populate G.nodes[*]["merged_from"] from graphify-out/dedup_report.json.

        Looks for dedup_report.json in:
        1. output_dir's sibling `graphify-out/` if output_dir is itself `<root>/graphify-out/obsidian`
        2. output_dir.parent directly (common graphify-out/ layout)
        3. `graphify-out/dedup_report.json` relative to cwd (fallback)

        Silently returns when the report is missing. Never raises.
        """
        candidates = [
            output_dir.parent / "dedup_report.json",
            output_dir / ".." / "dedup_report.json",
            Path("graphify-out") / "dedup_report.json",
        ]
        report_path = None
        for c in candidates:
            try:
                if c.exists():
                    report_path = c
                    break
            except OSError:
                continue
        if report_path is None:
            print(
                "[graphify] warning: --obsidian-dedup set but no dedup_report.json found",
                file=sys.stderr,
            )
            return
        try:
            data = json.loads(Path(report_path).read_text(encoding="utf-8"))
        except (json.JSONDecodeError, OSError) as e:
            print(f"[graphify] warning: could not read {report_path}: {e}",
                  file=sys.stderr)
            return
        # Build {canonical_id: [eliminated_ids]} from either merges[] or alias_map
        merges = data.get("merges", []) if isinstance(data, dict) else []
        for merge in merges:
            canonical_id = merge.get("canonical_id")
            if not canonical_id or canonical_id not in G:
                continue
            eliminated_ids = [
                e.get("id") for e in merge.get("eliminated", [])
                if isinstance(e, dict) and isinstance(e.get("id"), str)
            ]
            # Merge into existing merged_from (if dedup already ran into G.nodes)
            existing = G.nodes[canonical_id].get("merged_from") or []
            combined = sorted(set(existing) | set(eliminated_ids))
            if combined:
                G.nodes[canonical_id]["merged_from"] = combined
    ```
