## Milestone-Level Invariants Preserved

Every v1.4 phase preserved these invariants (audit confirmed):

- **D-02 MCP envelope** — `text_body + "\n---GRAPHIFY-META---\n" + json(meta)` with status codes on every new MCP tool
- **D-16 alias redirect** — all new handlers thread node IDs through `_resolve_alias`
- **D-18 compose don't plumb** — new MCP tools compose `analyze.py` / `delta.py` / `snapshot.py` primitives; only genuinely new analysis algorithms earn modules (`enrich.py`, `argue.py`)
- **`graph.json` read-only** — all new state in atomic `.tmp` + `os.replace` sidecars (`routing.json`, `manifest.json`, `enrichment.json`, `GRAPH_ARGUMENT.md`, `harness/*.md`)
- **`peer_id="anonymous"` default** — never derived from environment
- **`security.py` gate** — all new I/O routes through `validate_graph_path` + `sanitize_label*`
- **Snapshot double-nesting regression guard (Pitfall 20 / v1.3 CR-01)** — codified in Phase 18 via `snapshot.py::root` → `project_root` rename
